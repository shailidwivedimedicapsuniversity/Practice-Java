1.What is a Stack? 
A stack is a linear data structure that follows the Last In First Out (LIFO) principle,
means the last element added into stack, will be removed First.

2. main operations: 
1.push (to add an item in top of the stack) 
2.pop (to remove an item from top)
3.peek(see the top value of the stack). 
 
3. Uses ?
Stacks are used in various applications, like function call management(recursion)
