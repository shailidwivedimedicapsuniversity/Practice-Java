Def: 
LinkedList is a linear data structure that dynamically stores a list of data elements, where each element is called a node. Each node contains data and a reference (link) to the next node. The first node is called head, and the last node is called  tail.

Uses:
LL are flexible in size. Non-contigous memory
It is chain of elements 
They allow for efficient insertion and deletion of elements from any position, 
manipulation TC=O(1) , Search TC=O(n).
The LinkedList class in Java implements the List and Deque interfaces, providing various methods for manipulation. like: add, addFirst, addLast, remove ,removeFirst, removeLast size, isEmpty

Syntax: LinkedList<String> list = new LinkedList<>();

Types- 
1.Singular 2.Doubly 3. Circular
